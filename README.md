# Automated Dust Cleaning Bot using Arduino

## Overview

An autonomous dust cleaning robot developed using Arduino UNO for smart cleaning applications.

The robot is designed to move automatically, detect obstacles, avoid collisions, and perform dust cleaning using a suction mechanism. It provides a low-cost solution for basic automated cleaning systems.

## Features

- Autonomous robot movement
- Obstacle detection and avoidance
- Ultrasonic sensor based navigation
- Servo motor based direction scanning
- Four DC motor control
- Automatic speed adjustment based on distance
- Dust suction mechanism


## Hardware Components

- Arduino UNO
- Motor Driver Shield
- HC-SR04 Ultrasonic Sensor
- Servo Motor
- BO DC Motors
- Mini Drone Motor (Dust Suction)
- Battery Power Supply


## Software & Tools Used

- Arduino IDE
- Embedded C Programming


## Working Principle

1. The ultrasonic sensor continuously measures the distance of obstacles.
2. Arduino processes the sensor data.
3. If the path is clear, the robot moves forward.
4. When an obstacle is detected, the robot stops and moves backward.
5. Servo motor rotates the ultrasonic sensor to check left and right directions.
6. The robot chooses the best path and continues cleaning operation.
